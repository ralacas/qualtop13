SEGMENTACIÓN POR ETIQUETAS ESPECIALES

Segmentación por etiquetas especiales que permitan el Analisis de la disperción del subtotal.

Segmentación Etiquetas - Configuración


Las etiquetas Analiticas s deben configurar para llevar la segmentación de las facturas.

Segmentación Etiquetas - Historial


Al habilitar una etiqueta de segmentación, podremos visualizar el control utilizado en las facturas donde esta etiqueta fue empleada.

Asistente


Dentro de las Facturas Publicadas observaremos el boton "Segmentación" el cual nos abrira un asistente para:
  • Seleccionar las Etiquetas a Segmentar
  • Ingresaremos el Porcentaje o el Monto
  • Pulsamos Aplicar

Factura


Al generar la segmentación la factura guardara estos valores dentro de la pestaña Otra informaci&oacte;n y contiene la relacion de:
  • Tag
  • Fecha
  • Moneda
  • Porcentaje
  • Monto
  • Total (Subtotal de la Factura)
A manera de bitacora el sistema almacena:
  • Fecha de la ultima modificación
  • Usuario

Analisis de Segmentaciones


La información generada la podemos Analizar dentro del menu Contabilidad -> Informes -> Administración -> Análisis de Etiquetas Segmentación
  • Vista Lista
  • Vista Pivote
